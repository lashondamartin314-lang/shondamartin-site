This folder is for bootcamp PDFs and other downloadable resources.

To add a download:
1. Drop the PDF into this folder (e.g., "week-1-credit-foundations.pdf")
2. The corresponding button on the bootcamp page is already wired to look for these filenames:
   - week-1-credit-foundations.pdf
   - week-2-understanding-your-score.pdf
   - week-3-dispute-process.pdf
   - week-4-credit-profile.pdf
   - week-5-12-month-plan.pdf

If a PDF doesn't exist yet, the button still works but downloads the placeholder. Replace the placeholder with the real file when ready.

To use different filenames, edit bootcamp.html and change the href on each "Get Week X Documents" button.
